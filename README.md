# 我的世界｜段泽华个人网站

一个使用 Next.js、TypeScript、GSAP 和 Lenis 制作的中英双语个人网站。

## 本地运行

在项目目录右键运行 `start-site.ps1`，或在已配置 Node.js 与 pnpm 的终端执行：

```powershell
pnpm install
pnpm dev
```

浏览器打开 `http://localhost:3000`。

## 最常改的地方

- 个人文案与中英翻译：`components/Portfolio.tsx` 顶部的 `copy`
- 页面结构：`components/Portfolio.tsx`
- 颜色、排版与响应式样式：`app/globals.css`
- 图片：`public/images`
